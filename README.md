# Online Word Wolf

通話しながらワードウルフできる Web アプリ

## Functions

- user name
- user icon
- create room
- invite room
- join room
- theme card
- timer
- voting
- resalt
- continue

## Issue

- alertModal をつくる

- userthemeversion
- 認証の永続性を調査（"始める"時のみにしたい）
- 登録制も検討
