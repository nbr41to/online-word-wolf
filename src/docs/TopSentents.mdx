# ワードウルフとは？

ワードウルフは多人数で盛り上がる会話ができるゲームです。
あるお題について話をしてもらいますが、その中に一人違うお題が与えられた狼が混ざっています。
会話の中でその狼を見つけよう！

# なぜワードウルフ？

オンライン通話が日常的になった今、話題やネタが尽きたりしていませんか？
そんなときはこのアプリでワードウルフをして盛り上がりましょう！
